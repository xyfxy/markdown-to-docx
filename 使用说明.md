 # Markdown to DOCX 转换工具使用说明

## 项目简介

这是一个将Markdown文件转换为DOCX文档的Python工具，特别针对中文文档格式进行了优化。

## 环境要求

- Python 3.10+
- Windows/Linux/macOS

## 安装和设置

### 1. 创建虚拟环境

```bash
python -m venv venv
```

### 2. 激活虚拟环境

**Windows (PowerShell):**
```powershell
.\venv\Scripts\Activate.ps1
```

**Linux/macOS:**
```bash
source venv/bin/activate
```

### 3. 配置国内镜像源（可选）

```bash
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### 4. 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 基本用法

```bash
python main.py input.md output.docx
```

### 高级选项

```bash
python main.py input.md output.docx --font-size 16.0 --line-spacing 27.0 --verbose
```

### 参数说明

- `input.md`: 输入的Markdown文件路径
- `output.docx`: 输出的DOCX文件路径
- `--font-size`: 正文字体大小（点数）
- `--line-spacing`: 行间距（点数）
- `--template`: 可选的Word模板文件路径
- `--verbose, -v`: 启用详细输出和错误堆栈跟踪

### 示例

1. **基本转换:**
   ```bash
   python main.py test_sample.md output.docx
   ```

2. **自定义格式:**
   ```bash
   python main.py test_sample.md output.docx --font-size 12.0 --line-spacing 20.0
   ```

3. **使用模板:**
   ```bash
   python main.py test_sample.md output.docx --template template.docx
   ```

## 支持的Markdown元素

- 标题 (H1-H5)
- 段落
- 无序列表
- 有序列表
- 嵌套列表
- 粗体文本 (`**粗体**`)
- 斜体文本 (`*斜体*`)
- 中英文混合内容

## 输出格式特点

- 针对中文文档优化的字体设置
- 标准的公文格式
- 自动识别收件人格式
- 支持多级列表缩进
- 中英文字体分别处理

## 运行测试

```bash
python -m pytest tests/ -v
```

## 故障排除

1. **语法错误**: 确保所有Python文件末尾没有多余的markdown代码块标记
2. **依赖问题**: 确保所有依赖都已正确安装
3. **编码问题**: 确保Markdown文件使用UTF-8编码
