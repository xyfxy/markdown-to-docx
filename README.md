# Markdown转DOCX格式转换工具

## 项目概述

本项目是一个命令行工具，旨在将用户提供的Markdown格式文档转换为符合特定政府公文格式要求的DOCX文件。开发此工具的目标是自动化公文排版过程，提高效率，并确保输出文档的规范性和一致性，满足PRD（产品需求文档）中定义的排版标准。

## 主要特性

-   **标准Markdown解析**: 支持包括标题、段落、列表（有序与无序）、强调（加粗、斜体）等常见Markdown语法。
-   **公文格式DOCX生成**:
    -   严格按照预设的页面参数（A4纸张，特定页边距）创建文档。
    -   支持多级标题格式，包括特定的字体、字号、对齐方式和编号前缀。
    -   正文文本采用特定字体（如仿宋）、字号和行距，首行缩进。
    -   列表项具有规范的缩进和项目符号/编号。
    -   支持公文特有的“主送机关”格式。
-   **可配置格式选项**:
    -   通过命令行参数调整正文的行距和字号。
    -   支持使用自定义Word模板文件（`.docx`）作为基础样式。
-   **命令行界面**: 提供简单易用的命令行接口，方便集成和脚本化操作。
-   **中文字符与字体支持**:
    -   完美处理中文字符。
    -   针对中文、英文和数字分别应用指定字体（如中文使用仿宋、黑体、楷体等，英文和数字使用Times New Roman）。
-   **详细日志与错误处理**: 提供详细的执行日志和清晰的错误提示，支持verbose模式输出调试信息。
-   **模块化设计**: 代码结构清晰，分为Markdown解析、DOCX生成和命令行接口三大模块。
-   **可测试性**: 包含单元测试和集成测试，确保代码质量和功能稳定性。

## 技术栈

-   Python 3.8+
-   **核心库**:
    -   `python-docx`: 用于创建和操作DOCX文件。
    -   `markdown`: 用于将Markdown文本转换为HTML。
    -   `click`: 用于构建命令行界面。
-   **测试库**:
    -   `pytest`: 用于执行单元测试和集成测试。
    -   `pytest-mock`: (或 `unittest.mock`) 用于模拟对象依赖。

## 使用方法

通过 `main.py` 脚本使用本工具。

1.  **基本命令**:
    ```bash
    python main.py <input_markdown_file> <output_docx_file>
    ```

2.  **示例**:
    ```bash
    python main.py my_document.md my_document.docx
    ```

3.  **命令行选项**:
    -   `INPUT_MD`: (必需) 输入的Markdown文件路径。
    -   `OUTPUT_DOCX`: (必需) 输出的DOCX文件路径。
    -   `--line-spacing FLOAT`: 设置正文行距 (单位：磅, 例如 `27`)。
    -   `--font-size FLOAT`: 设置正文字号 (单位：磅, 例如 `16` 对应三号字)。
    -   `--template FILE_PATH`: 指定一个可选的Word模板文件（`.docx`）的路径。工具将基于此模板创建文档，并继承其样式。
    -   `--verbose` / `-v`: 输出详细的执行日志，便于问题排查和调试。
    -   `--help`: 显示帮助信息并退出。

4.  **示例Markdown输入**:
    ```markdown
    # 文档标题示例
    
    这是一个普通段落，包含一些**加粗**和*斜体*文本。
    
    ## 一级标题示例
    
    - 无序列表项1
    - 无序列表项2
    
    ### 二级标题示例
    
    1. 有序列表项1
    2. 有序列表项2
    ```

## 输出示例说明

生成的DOCX文件将严格遵循PRD（产品需求文档）中定义的公文格式规范。这包括但不限于：

-   **页面设置**: A4纸张，特定的页边距（上3.7cm，下3.5cm，左2.8cm，右2.6cm）。
-   **标题格式**:
    -   文档主标题（大标题）：方正小标宋简体，二号字（22pt），居中，特定行距。
    -   一级标题（如“一、”）：黑体，三号字（16pt）。
    -   二级标题（如“（一）”）：楷体，三号字（16pt）。
    -   三级、四级标题：仿宋，三号字（16pt）。
-   **主送机关**: 顶格，仿宋，三号字，末尾加全角冒号。
-   **正文格式**: 仿宋，三号字（16pt），固定行距27磅，两端对齐，首行缩进2字符。中文字体为仿宋，英文、数字为Times New Roman。
-   **列表**: 项目符号和编号，具有规范的缩进。
-   **强调**: 加粗文本使用仿宋加粗，斜体文本使用仿宋斜体。

用户可以通过命令行选项调整正文的字号和行距，或提供自定义Word模板以进一步定制样式。
